Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is here :
1. innerHTML - sets or gets HTML content as a string ;
2. innerText - gets or sets visible text while ignoring hidden elements;
3. textContent - retrieves or sets all text, including hidden elements, without parsing HTML.

Question-3: What is event delegation in the DOM?

In the DOM event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements using event bubbling.
